# Marmalade
Интернет-магазин для продажи натурального мармелада с акцентом на визуал и удобство. Ключевые функции: красивый каталог с фильтрацией по вкусам, конструктор подарочных наборов и упрощённое оформление заказа. Администратору доступна интуитивная панель для управления товарами и отслеживания заказов. Проект реализован на современном стеке технологий для обеспечения высокой производительности и пользовательского опыта.
